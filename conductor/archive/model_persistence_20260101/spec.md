# Specification: Model Persistence\n\n## Overview\nThis track implements the capability to save and load trained neural network models to and from disk using a minimalist, custom binary format. This avoids dependencies on complex serialization libraries and ensures fast loading on embedded devices.\n\n## Binary Format Schema\n- **Magic Bytes:** 4 bytes (e.g., 'HWMD' for HotWord MoDel).\n- **Version:** 2 bytes (uint16) for future-proofing.\n- **Metadata:** \n  - Input size (uint32).\n  - Number of weights (uint32).\n  - Number of bias units (uint32).\n- **Data:**\n  - Weights data (float32 array).\n  - Bias data (float32 array).\n\n## Core Functions\n- **SaveModel:** Serializes the weights and biases of a model into the specified file path.\n- **LoadModel:** Reads a model file, validates the format, and restores the weights and biases.\n\n## Error Handling\n- Detect and report invalid magic bytes or unsupported versions.\n- Handle I/O errors during file operations.
