# Specification: Professional CLI (Cobra/Viper)\n\n## Overview\nThis track replaces temporary verification scripts with a structured, professional Command-Line Interface (CLI). It uses the industry-standard Cobra and Viper libraries to handle commands, flags, and configuration files.\n\n## CLI Structure\n- **root:** The base `hotword` command.\n- **train:** Handles the training pipeline (`hotword train --data <dir> --out <model>`).\n- **verify:** Tests a saved model against a dataset (`hotword verify --model <file> --data <dir>`).\n- **version:** Displays build and version information.\n\n## Configuration (Viper)\n- Support for configuration files (e.g., `config.yaml`).\n- Flags should override configuration file values.\n- Key config items: `sample_rate`, `mel_filters`, `learning_rate`, `detection_threshold`.\n\n## User Experience\n- Use colorful and formatted output for progress and results.\n- Provide clear help text and usage examples for every command.
